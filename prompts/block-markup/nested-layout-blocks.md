# Build A Two-Service Overview Page

Can you create or update a WordPress page titled **Community Services Overview**
with a simple two-column services layout?

I need a compact page that introduces two community services side by side. The
layout should feel like one coherent section with two balanced service summaries.

Content requirements:

- A short page introduction section.
- Two side-by-side service summaries.
- Each service summary needs its own heading and one paragraph of explanatory
  copy.

Make the page easy for a WordPress site owner to rearrange or edit visually in
the editor.
