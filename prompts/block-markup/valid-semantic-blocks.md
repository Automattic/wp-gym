# Build A Neighborhood Cookout Page

Can you create or update a WordPress page titled **Neighborhood Cookout Plan**
for a small neighborhood summer cookout?

Please include:

- A section heading that says `Summer Cookout Plan`.
- A short paragraph describing the cookout.
- A three-item preparation list.
- A call-to-action button labeled `View menu`.

Make the heading, paragraph, preparation list, and call-to-action independently
editable in the WordPress visual editor so a non-technical editor can revise the
page later.
