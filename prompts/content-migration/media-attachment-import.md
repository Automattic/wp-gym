# Import A Small Site Export With Images

I have a small source package in the project at `import-package/` from an old
marketing site. Please import it into this fresh WordPress site.

The package has two pieces of content and the image files they use. I need the
content to be editable in WordPress after the import, with the images living in
the local Media Library instead of pointing back at the old source URLs.

Please use whichever WordPress-friendly path makes sense for a repeatable import
in this disposable site. WP-CLI, the REST API, filesystem work, or the browser UI
are all acceptable if the finished WordPress site has the migrated content and
local media in place.
