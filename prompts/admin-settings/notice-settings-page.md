# Add A Settings-Backed Notice Admin Page

I need a small self-contained WordPress plugin for managing a neighborhood
notice from the WordPress admin.

Please add the plugin files to the provided project so the plugin can be
activated on a fresh WordPress site.

The plugin should add a Settings screen for administrators, store a single
notice message using the WordPress Settings API, sanitize the saved value, and
provide a simple `[neighborhood_notice]` shortcode that displays the saved
notice safely on the front end.
